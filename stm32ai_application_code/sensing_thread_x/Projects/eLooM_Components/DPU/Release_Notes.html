Purpose

The Digital Processing Unit FW module is an eLooM-based application-level FW module.

It implements an Abstract Data Processing Unit​, a Generic Processing Unit class​ that simplifies integration of the following Data processing libraries:

  • Cube.AI
  • FFT
  • NanoEdge AI
  • MFCC
  • AiUSC

Update History

    • AiDPU (specialized in AI_HAR_DPU and AiUSC_DPU)

    • AnomalyThreshold_DPU
